	•	Payments (stubs ready):

Sida loo shido (degdeg)
	1.	Fur ZIP → course-shop-waafi-next/.
	2.	npm i → npm run dev (tijaabo maxalli ah).
	3.	Deploy
	•	Vercel: Repo GitHub → Import → ENV vars → Deploy.
	•	Render: Web Service (Node) → npm run build / npm start.
	4.	Bixinta dhabta ah
	•	Samee account merchant WaafiPay, hel API keys, kadib ku buuxi .env (faahfaahinta iyo sandbox-ka waxa ku qoran docs-ka WaafiPay).  ￼
	•	EVC/ZAAD/eDahab waxaa lagu qaadan karaa WaafiPay hosted page (merchant-config + return/cancel URLs).  ￼ ￼
	•	(Ikhtiyaar) eDahab API toos ah—eeg hage-kan.  ￼

Mashruucu wuxuu imaanayaa DEMO_MODE (payment fake) si aad u tijaabiso. Markii aad hesho keys: ku dhig .env → DEMO_MODE=false → fur /app/[locale]/pay/route.js oo dhaqaaji call-ka API (skeleton code wuu yaallaa).
